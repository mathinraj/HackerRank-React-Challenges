

# HackerRank React Challenges Solutions  
  
Welcome to this repository! This is a collection of solutions to **React challenges** available on [HackerRank](https://www.hackerrank.com/domains/react). These solutions are intended to help those who are struggling with these challenges or want to learn React by studying practical examples.

## About  
  
HackerRank provides a variety of React challenges to help developers practice and improve their skills. However, finding solutions to these challenges can be difficult, especially for beginners. This repository aims to provide clear and well-documented solutions to these challenges, making it easier for learners to understand and implement React concepts.  
  
---  
  
## Challenges Covered  
  
Below is the list of React challenges from HackerRank that are solved in this repository:  
  
1. **Article Sorting** -   Creating a React application called "Article Sorting" that displays a list of articles and allows users to sort them based on upvotes and publication date.

2. **Blog Post** - Creating a React application called "Blog Post" that allows users to create, display, and delete blog posts.

3. **Code Review Feedback** - Creating a React application called "Code Review Feedback" that tracks and manages feedback on various aspects of code quality.

4. **Contact Form** - Creating a simple React application called "Contact Form" that collects user information and displays it below the form upon submission.

5. **CryptoRank Exchange** - Developing a React application called "CryptoRank Exchange" to estimate the number of cryptocurrency coins a user can receive based on an entered amount of fiat currency.

6. **Employee Validation** - Creating a React app that helps employees validate and update their existing information through a survey form.

7. **Item List Manager** - Creating a simple React application called "Item List Manager" that displays a list of items and allows users to add new items to the list. The items will be displayed in an unordered list (`<ul>`), and there will be an input field along with a button to add new items to this list.

8. **Patient Medical Records** - Creating a React application called "Patient Medical Records" that allows users to view and cycle through patients' medical records.

9. **Slideshow** - Creating a simple React application called "Basic Slideshow" that displays a series of slides and allows users to navigate through them.

10. **Word Omitter** - Creating a simple React application called "Word Omitter" that dynamically filters out specific words from the user input in real-time.
  
  
Each challenge has its own folder containing:  
- Problem Statement
- Solution code

## Contributing  
  
Contributions are welcome! If you have a better solution or want to add solutions to more challenges, feel free to open a pull request. Please ensure your code is well-documented and follows best practices.  
  
1. Fork the repository.  
2. Create a new branch for your changes:  
```bash  
git checkout -b your-branch-name  
```  
3. Commit your changes:  
```bash  
git commit -m "Add solution for Challenge Name"  
```  
4. Push your changes to your forked repository.  
5. Open a pull request with a detailed description of your changes.
